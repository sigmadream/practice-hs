# TypeClass

- Haskell은 타입과 정의를 분리함
- 타입 클래스는 Java의 인터페이스/추상 클래스와 유사
- Haskell은 과부하를 지원하지 않음

## basic type classes

- Enum
- Eq
- IO
- Integral
- Num
- Ord
- Read
- Show
- Functor
- Monad
- Monoid

### Show

Show 클래스의 인스턴스들은 show 함수를 이용하면 String 형태로 출력됩니다.

```haskell
ghci> show 3
"3"
ghci> show 5.33
"5.33"
ghci> show True
"True"
ghci>
```
